Site Architecture Overview

This portfolio is built as a modular, data‑driven system using GitHub Pages and Jekyll. The site is structured for maintainability, scalability, and clarity—similar to a lightweight production documentation platform.

  • Componentized Layout: Header, footer, résumé cards, and skill sections are rendered through reusable HTML includes.
  • YAML‑Driven Content: Résumé, skills, experience, and use cases are generated from structured data files.
  • Centralized Design System: All styling is managed through a custom theme (neubyte.css) with tokens for color, spacing, and typography.
  • Documentation Structure: Architecture, BRD, FS, API, and Use Case documents are organized into dedicated folders.
  • Print‑Optimized Résumé: A dedicated print stylesheet enables clean PDF export directly from the browser.
  • Static Hosting Pipeline: GitHub Pages automatically builds and deploys the site using Jekyll.
